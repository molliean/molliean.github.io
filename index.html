Common CSS Properties

5 Most Common Properties

Property Description
color Specifies the text color of the contents of an HTML element
font-size Specifies the size of the text in pixels (px)
background-color Specifies the background color of an HTML element
height and width Specifies the height and width of an HTML element in pixels
border Allows you to set a border for an HTML element

Other Important Properties

Property Description
display Specifies the display behavior of an HTML element
padding Specifies the space between an element and its border
margin Specifies the space between an element's border and other elements
border-radius Defines an HTML element's borders, can also add rounded corners
line-height Specifies the height of a line
